Lists

Unordered Lists

  • Use the ul tag to start an unordered list.
  • To add an item, use the li tag.
  • Unordered lists can be used to list items that have no direct ordering.
    • You can...
    • even have...
    • nested lists!!!

Ordered Lists

  1. Use the ol tag to start an ordered list.
  2. To add an item, use the li tag like in the unordered list.
    1. Yet again...
    2. you can nest lists!!!
      • Nested...
      • Nested...
      • Lists!!!
